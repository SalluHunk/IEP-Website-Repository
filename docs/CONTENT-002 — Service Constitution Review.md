# OPERATION PHOENIX

## CONTENT-002 — Service Constitution Review

Mission Type

Editorial Architecture Review

Purpose

Review the nine synthesized service documents as one integrated consulting portfolio.

This mission does NOT generate new content.

This mission critiques, refines and aligns the existing service documents.

--------------------------------------------------

Repository Sources

Use:

CONTENT-001

All nine Service documents

MASTER-SERVICE-MATRIX

Homepage Narrative Constitution

PDC-001

CRN-001

--------------------------------------------------

Objectives

Review the service catalogue as though you are an independent consulting partner reviewing IEP's market positioning.

Focus on:

• clarity

• differentiation

• narrative flow

• commercial positioning

• consistency

--------------------------------------------------

For EACH service evaluate:

1.

Purpose

Can the reader immediately understand what this service provides?

--------------------------------------------------

2.

Differentiation

Does this service overlap with another?

If yes:

identify the overlap

recommend boundary clarification

--------------------------------------------------

3.

Commercial Position

Would a client understand:

Why buy this?

When buy this?

Expected outcome?

If not,

identify the weakness.

--------------------------------------------------

4.

Evidence Integrity

Does every major claim remain supported?

Identify:

Verified

Partial

Weak

--------------------------------------------------

5.

Terminology

Ensure terminology is consistent across all services.

Examples:

Opportunity

Assessment

Diagnostic

Review

Feasibility

Design

Innovation

Assurance

Monitoring

Avoid unnecessary synonym drift.

--------------------------------------------------

Portfolio Review

Evaluate the nine services together.

Answer:

Does the portfolio feel complete?

Is anything duplicated?

Is anything obviously missing?

Does the order tell a logical consulting story?

Would you reorder any services?

Explain why.

--------------------------------------------------

Homepage Alignment

Confirm every service supports the Homepage Narrative Constitution.

Identify any conflicts.

--------------------------------------------------

Output

Create:

CONTENT-002-Service-Constitution-Review.md

Include:

Executive Review

Service-by-Service Findings

Portfolio Findings

Recommended Improvements

Priority Actions

No rewriting of the service content.

Only recommendations.

--------------------------------------------------

Repository Rules

No marketing embellishment.

No invented capabilities.

No unsupported positioning.

Maintain evidence integrity.

Stop.
